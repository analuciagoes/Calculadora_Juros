# Calculadora de Juros

Programa console em C# que calcula juros de mora com base em um valor principal e data de vencimento.

## Funcionalidade

- Calcula juros de mora de 2,5% ao dia para pagamentos em atraso
- Valida entradas de valor e data
- Exibe resultado detalhado com valores calculados

## Pré-requisitos

- .NET 8.0 SDK
